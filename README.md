# Spotify-1 2023 Dataset 🎵

## 📌 Overview
This repository contains the **Spotify 2023 dataset**, which includes detailed information about tracks released or popular in 2023. The dataset is designed for **data analysis, machine learning, and visualization projects** related to music trends, audio features, and artist popularity.

![spotify image]("./preview1.png")




## 📂 Dataset Contents
The dataset includes the following key attributes:

- **Track Information**
  - Track Name
  - Artist(s)
  - Album
  - Release Date
  - Popularity Score

- **Audio Features**
  - Danceability
  - Energy
  - Loudness
  - Speechiness
  - Acousticness
  - Instrumentalness
  - Liveness
  - Valence
  - Tempo

- **Metadata**
  - Duration (ms)
  - Key
  - Mode
  - Time Signature
  - Genre (if available)

## 🎯 Objectives
This dataset can be used for:
- Analyzing **music trends in 2023**
- Building **recommendation systems**
- Exploring **relationships between audio features and popularity**
- Visualizing **genre distributions and artist dominance**
